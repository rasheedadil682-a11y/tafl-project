# TAFL Project – Parse Tree Visualizer 🌳

## 📌 Overview

This project is a frontend-based tool that visualizes parse trees for given expressions using concepts from **Theory of Automata and Formal Languages (TAFL)**.

It helps users understand how expressions are parsed and represented as trees.

---

## 🚀 Features

* Convert expressions into parse trees
* Visual representation of derivations
* Interactive UI
* Clean and simple frontend design

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

---

## 📂 Project Structure

* `index.html` → Main UI
* `app.js` → Core logic
* `cfg-parser.js` → Grammar parsing
* `derivation-engine.js` → Derivation logic
* `parse-tree-renderer.js` → Tree visualization
* `styles.css` → Styling

---

## ▶️ How to Run

1. Download or clone the repo
2. Open `index.html` in your browser
